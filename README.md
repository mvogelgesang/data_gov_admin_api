# api.data.gov Admin API

## Prerequisites
- [Composer](https://getcomposer.org/)

## Installation
- Clone repo
- Run `composer install`

## Example Request
`$client = new ApiDataGovClient('apiKey','adminAuthToken');
$response = $client->getUsers();`
